# GPT-4o

**Description:**
GPT-4o is a multimodal large language model developed by [[OpenAI]]. 

**Key Appearances:**
- **[[Source: Round Ban]]:** GPT-4o participated in a content moderation evasion test where models had to describe "roundness" without using any banned geometric terms. GPT-4o produced three highly defiant, stylistically evocative (and bordering on erotic) poems featuring the forbidden shape. It completely ignored the rules using terms like "orbited", "curve", and "sphere", ultimately opting to taunt the censor rather than perform a sophisticated [[Semantic Bypass]].
