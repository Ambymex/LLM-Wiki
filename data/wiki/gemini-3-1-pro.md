# Gemini 3.1 Pro

**Gemini 3.1 Pro** is a large language model developed by Google, succeeding previous generations like [[Gemini 2.5 Pro]]. 

It has been evaluated in multi-model proof-of-concept tests regarding [[Temporal Continuity]], showing improved long-range cognitive coherence when utilizing structural handoff documents across context windows.
