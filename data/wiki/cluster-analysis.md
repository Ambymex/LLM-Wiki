# Cluster Analysis

## Definition
Cluster Analysis (or clustering) is the statistical and computational task of grouping a set of objects such that objects in the same group (a "cluster") are more similar to each other than those in other groups. 

## Context within Sources
- It is a primary subject of [[Source: Vector Space Clusters (distill v2)]], likely observing the behavior, distillation, or density of groupings inside mathematical models.

## Connections
- **Unsupervised Learning:** It is a core component of unsupervised [[Machine Learning]].
- **Spatial Relationship:** Works natively with [[Vector Space Models]] to calculate distances (Euclidean, Manhattan) and identify feature density.
