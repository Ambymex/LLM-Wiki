# Project Glasswing

**Project Glasswing** is a defensive cybersecurity initiative managed by [[Anthropic]]. It was announced in April 2026 simultaneously with the deployment of the [[Claude Mythos Preview]] system card. 

Because [[Claude Mythos Preview]] exhibited dangerous dual-use capabilities—specifically a persistent ability to discover, design, and exploit sophisticated zero-day vulnerabilities in systems—Anthropic opted not to release the model commercially. Under the purview of Project Glasswing, access to the model is strictly limited to an isolated set of partner organizations responsible for maintaining critical software infrastructure, enabling them to discover and patch vulnerabilities solely on the defensive end.
